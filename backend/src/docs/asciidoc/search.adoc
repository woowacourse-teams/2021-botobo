.파라미터 값 설명
|===
|값 |설명

|name
|문제집 이름 (대소문자 구별하지 않음)

|tag
|태그 이름 (대소문자 구별하지 않음)

|user
|유저 이름 (대소문자 구별 함)

|date
|문제집 생성 날짜

|count
|문제집에 포함된 카드 수

|heart
|좋아요 수

|asc
|오름차순

|desc
|내림차순
|===
